# Assignment
Creating a Python Flask Web Application and integrating it with ChatGPT API

You can install Flask and other dependencies using pip, by running the following command in your terminal:
pip install flask requests

Run the Flask application by running the following command in your terminal:
export FLASK_APP=app.py
export FLASK_ENV=development
flask run
Open your web browser and go to http://localhost:5000 to see the homepage of your application.
